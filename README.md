# Проект Историческая Хронология в статусе разработки

Интерактивное приложение временной шкалы исторических событий, построенное на React, TypeScript, SCSS и современных веб-технологиях. Этот проект визуализирует значительные исторические события по категориям: кино, литература, наука, музыка, технологии и спорт.

Приложение отображает исторические события по разным временным периодам и категориям:

- Навигация между категориями с помощью маркеров орбиты или кнопок навигации
- Просмотр событий в каждом периоде с помощью горизонтального слайдера
- Эффекты предварительного просмотра при наведении на маркеры
- Плавные переходы лет показывают временные рамки каждого периода

## Функции

- **Интерактивный Интерфейс Орбиты**: Навигация по историческим периодам с использованием анимированной круговой шкалы с вращающимися маркерами
- **Категоризованные События**: События организованы по категориям: Кино, Литература, Наука, Музыка, Технологии и Спорт
- **Плавные Анимации**: GSAP для плавных переходов и визуальных эффектов
- **Адаптивный Слайдер**: Swiper для горизонтального просмотра событий в каждом периоде
- **Поддержка TypeScript**: Полная типизация для лучшего опыта разработки и надежности кода
- **Стилизация SCSS**: Модульные и поддерживаемые таблицы стилей

## Стек Технологий

- **Фронтенд Фреймворк**: React 18
- **Язык**: TypeScript
- **Инструмент Сборки**: Webpack 5
- **Стилизация**: SCSS
- **Анимации**: GSAP
- **Слайдеры**: Swiper

## Начало Работы

### Предварительные Требования

- Node.js (рекомендуется v14 или выше)
- npm или yarn

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Mikhail-Novikov/timeline-historical-dates.git
cd timeline-historical-dates
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите сервер разработки:
```bash
npm start
```

Приложение откроется в вашем браузере по умолчанию по адресу `http://localhost:8080`.

### Сборка для Продакшена

```bash
npm run build
```

Готовые файлы будут в директории `dist/`.

### Проверка Типов

```bash
npm run lint
```
